# Weather Dashboard

## Purpose
A weather dashboard that allows you to see the current and future weather conditions and creates a search history for multiple cities using Open Weather Map API

## Built with 
* Open Weather Map API
* BootStrap
* Open Iconic
* Google Fonts
* JavaScript
* HTML
* CSS
